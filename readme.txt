== Installation ==

You can install *WP Activity Streaming* directly from the WordPress admin! Visit the *Plugins/Add New* page and search for '*WP Activity Streaming'. Click to install.

Once installed and activated visit the WPtouch admin page (*Settings/*WP Activity Streaming*) to customize your WPtouch appearance.

== Frequently Asked Questions ==

= Why doesn't my feed from iTunes work 100% correctly? =

Apple's feeds are notoriously broken. We have some logic in there to try and fix them, but if yours still isn't working, please let us know so we can work on it some more.
